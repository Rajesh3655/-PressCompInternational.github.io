# Press Comp International Website

A premium, high-conversion industrial B2B website for **Press Comp International** — India's leading precision stamping and tooling manufacturer since 1983.

## 🌐 Live Site
[https://rajesh3655.github.io/-PressCompInternational.github.io/](https://rajesh3655.github.io/-PressCompInternational.github.io/)

## 📄 Pages
- **Home** — Hero, Trust bar, About snapshot, Capabilities, Industries, Why PCI, Testimonials, CTA
- **About** — Company story, Milestones timeline, Mission/Vision, Leadership
- **Capabilities** — Stamping, Tooling, Reel-to-Reel, Custom Engineering
- **Products** — Filterable product grid by industry
- **Industries** — Telecom, Automotive, Electronics, Electrical deep-dives
- **Quality** — Certifications (IATF 16949, ISO 9001), Awards, QA process, KPIs
- **Gallery** — Filterable photo gallery with lightbox
- **Contact** — Enquiry form (Coming Soon), Contact info, Google Maps

## ✨ Features
- 🎨 Premium dark industrial design (Navy, Blue, Metallic)
- 📱 Fully responsive — mobile first
- 💬 WhatsApp float button (quick enquiry)
- 📊 Animated counters & scroll reveal animations
- 🏷️ Demo site banner linking to SoftSite Solutions
- 📬 Coming Soon email form with redirect to softsitesolution.in
- ⚙️ SEO optimised (meta tags, semantic HTML, canonical links)
